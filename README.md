<br/></br>### 课程介绍:
本课程所讲内容是AR技术现今订制中的常见案例，其中基础模板是2016工信部举办的CES中美创客大赛的获奖案例。
另外针对项目中客户经常会要求到的几个功能进行讲解，例如社会化分享（微信，微博分享）。IOS端除了上架苹果商店外的几种解决方案。宣传片AR化演示等。
本套课程为零基础可学课程。课程将会分为三个部分：1思路讲解。2流程讲解与重点问题解释。3从零逐步完成。来满足不同基础同学的需求。
